#  晴空 ClearSky

> 关心你的每一次情绪波动  
> 一款面向校园的心理健康服务 App

---

##  项目简介

**晴空 ClearSky** 是大三上学期软件工程实践的小组作业，作为一个高校学生打造的心理健康服务平台，集 **心理咨询、情绪测评、知识资源、社区互动** 于一体。  
项目旨在通过数字化手段，帮助学生更方便地进行自我情绪管理与心理支持。

##  核心功能

| 模块 | 功能说明 |
|------|-----------|
|  AI心理陪伴| 仿聊天界面，用户输入文本，AI实时回复|
|  心理咨询预约 | 展示咨询师列表，支持按专业筛选与一键预约，查看可约时间段 |
|  情绪测评 | 提供焦虑、抑郁、压力等量表测评，完成后生成图表化报告（含雷达图与综合评分） |
|  心理资源库 | 汇集心理学课程、文章与视频资源，可收藏、分类浏览 |
|  校园社区 | 用户可发布帖子、上传图片、互动评论，营造支持性社群氛围 |
|  个人中心 | 展示用户信息、测评报告、预约记录，并支持修改资料与退出登录 |
